Guide

☆) Add: first input words you want to search your subject with.
Second input the subject title, can't make duplicate with the exact same name.
Last input is the description you want to write for your subject.
☆) Search bar: type as close for the subject as you want, will search with subject text & search references you typed for this subject
☆) Import: input JSON file with the data you want to add.
☆) Export: download to your pc whole table with data for easy sending to other & easy importing.
☆) : load subject content into add window for editing.
☆) : deletes subject row & description row.
☆) For italic use: #i# & #/i# instead of: <i> & </i>
☆) For bold use: #b# & #/b# instead of: <b> & </b>
☆) For underline use: #u# & #/u# instead of: <u> & </u>
☆) Input that will be deleted on entry: ', ", <i>, </i>, <b>, </b>, <u>, </u>
